## Purpose
Handoff documents are used to transfer context between agents or sessions when a task is interrupted or requires a specific mode switch (e.g., Investigation -> Implementation).

## Naming Convention
`YYYY-MM-DD-<TYPE>-<slug>.md`

## Handoff Types
- **TASK:** Basic task handoff
- **INVESTIGATION:** Results of an investigation
- **INQUIRY:** Deep dive questions
- **BUG:** Bug report and context
- **DECISION:** Architectural decision context
